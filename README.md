# Battleships - Jetpack Compose

An online multiplayer Battleship game for Android, built entirely with Kotlin and Jetpack Compose.

## About

This project is a modern Android implementation of the classic Battleship board game. Players can compete against each other in real-time multiplayer matches, placing their ships strategically and attempting to sink their opponent's fleet.

## Features

- **Online Multiplayer** - Play against other players in real-time
- **Modern UI** - Built with Jetpack Compose for a fluid, responsive experience
- **Classic Gameplay** - Traditional Battleship rules and mechanics

## Tech Stack

- **Language:** Kotlin
- **UI Framework:** Jetpack Compose
- **Architecture:** Android modern architecture components

## Getting Started

### Prerequisites

- Android Studio (latest stable version recommended)
- JDK 11 or higher
- Android device or emulator running API 21+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MortezaSaf12/Battleships-Jetpack-Compose.git
   ```

2. Open the project in Android Studio

3. Sync Gradle dependencies

4. Build and run the app on your device or emulator

## How to Play

1. Launch the app and connect to a game
2. Place your ships on the grid
3. Take turns firing at your opponent's grid
4. The first player to sink all enemy ships wins
